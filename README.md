# Mini Finance

## Descripción
Simulador de presupuesto personal para registrar ingresos, gastos y metas de ahorro. El usuario puede visualizar su saldo y recibir advertencias según sus hábitos de gasto.

## Idea elegida
Mini Finance — Idea 5 del proyecto integrador.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades principales
- Registro de ingresos y gastos
- Clasificación de gastos por categoría
- Cálculo de saldo disponible
- Alertas visuales cuando se supera un límite de gasto
- Guardado de datos con localStorage

## Estructura del proyecto
- `index.html` — Página de inicio
- `pages/simulador.html` — Simulador de ingresos y gastos
- `pages/resumen.html` — Resumen financiero
- `pages/market-rates.html` — Cotizaciones
- `base/styles/` — Archivos CSS
- `assets/` — Imágenes y recursos
